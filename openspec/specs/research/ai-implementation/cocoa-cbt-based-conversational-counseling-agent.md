# CoCoA：基于CBT的对话咨询智能体

## 使用认知扭曲专用记忆和动态提示

**作者**：Suyeon Lee, Jieun Kang, Harim Kim, Kyoung-Mee Chung, Dongha Lee, Jinyoung Yeo

**来源**：延世大学人工智能系、心理学系

**发布时间**：2024年2月
COCOA: CBT-based Conversational Agent (https://arxiv.org/abs/2402.17546)

---

## 摘要

对提供心理保健的对话智能体的需求持续增加。在本工作中，我们开发了一个称为CoCoA的心理咨询智能体，它应用认知行为疗法（CBT）技术来识别和解决来访者陈述中固有的认知扭曲。具体而言，我们构建了一个记忆系统，以有效管理咨询所需的信息，同时从来访者的陈述中提取高水平的洞察。此外，为了确保咨询智能体生成适当的响应，我们引入了动态提示，以灵活地应用CBT技术并促进信息的适当检索。我们让CoCoA与Character.ai的角色进行对话，创建了评估数据集。然后，我们请GPT评估构建的咨询数据集，我们的模型表现出与其他模型统计学上的显著差异。

---

## 1. 引言

在当今世界，越来越多的人表达对心理健康的担忧。因此，对解决这些问题的心理辅导需求更高。然而，有些人由于成本和社会耻辱等因素无法获得适当的心理健康护理。在这种情况下，用于心理咨询的对话智能体可以为那些对传统心理健康服务犹豫不决的人来说是一个有价值的替代方案。此外，它可以通过增强可及性和便利性来有效满足日益增长的心理健康需求。

最近的研究开发了利用对话智能体改善精神状态的方法，如移情对话生成和情绪支持对话。然而，上述研究专注于提供情绪支持，而不是推进解决根本心理问题。为了解决这个问题，我们提出了一个基于LLM的咨询师智能体，它利用CBT技术提供心理咨询。CBT技术旨在识别和转换认知扭曲，即负面情绪的根本原因。

为了利用CBT技术提供类似人类的咨询，我们提出了几种方法。咨询师需要从与来访者的对话中提取相关信息以生成响应。为了有效地从来访者的陈述中提取、存储和检索高级信息，我们将记忆结构化为两个组件：用于存储通过咨询学习的个人信息的基本记忆，以及用于存储从陈述中识别的认知扭曲的认知扭曲记忆。此外，咨询师在响应之前会根据来访者的陈述规划要解决的认知扭曲并选择CBT技术及其相应的解决阶段。我们动态组装提示，使我们的咨询智能体经过此过程后生成响应。因此，我们引导智能体在生成咨询话语之前经过中间步骤以进行思维链。

为了评估我们智能体的表现，我们在CoCoA与模拟知名人物的生成智能体之间进行咨询会话。我们通过Character.ai服务选择了8位知名人物的模拟角色，如梵高或杰伊·盖茨比。然后，我们指示GPT-3.5根据五个标准评估构建的数据集。

## 2. 相关工作

### 2.1 对话智能体用于心理健康

对话智能体在心理健康领域的研究可以大致分为两类：移情对话生成和情绪支持对话。移情对话生成旨在生成理解用户情绪并适当回应的对话。情绪支持对话旨在通过提供情感支持来帮助用户。然而，这些方法侧重于提供情感支持，而不是解决根本的心理问题。

### 2.2 认知行为疗法（CBT）

CBT是一种心理治疗方法，旨在通过识别和改变负面思维模式来改善心理健康。CBT基于认知模型，该模型提出思想、情感和行为相互关联。CBT技术包括认知重构、行为实验和暴露疗法等。

### 2.3 记忆增强的对话智能体

记忆增强的对话智能体使用外部记忆存储和检索相关信息。这些方法可以有效地管理长期对话历史并生成连贯的响应。然而，现有的记忆系统通常设计用于一般对话，而不是专门用于心理咨询。

## 3. 方法

### 3.1 系统概述

CoCoA系统由三个主要组件组成：

1. **记忆系统**：存储和检索咨询相关信息
2. **认知扭曲识别模块**：识别来访者陈述中的认知扭曲
3. **动态提示生成模块**：根据当前对话状态生成适当的提示

### 3.2 记忆系统

记忆系统分为两个部分：

#### 基本记忆

- 存储来访者的个人信息（如姓名、年龄、职业）
- 存储咨询过程中学习到的关键信息
- 使用向量数据库进行高效检索

#### 认知扭曲记忆

- 专门存储识别的认知扭曲
- 每个认知扭曲条目包括：
  - 扭曲类型（如全有或全无思维、过度概括）
  - 原始陈述
  - 识别时间
  - 解决状态（未解决、正在解决、已解决）

### 3.3 认知扭曲识别

使用预训练的LLM识别来访者陈述中的认知扭曲。识别过程包括：

1. 提取陈述中的关键短语
2. 匹配已知的认知扭曲模式
3. 评估扭曲的严重程度
4. 确定优先解决的顺序

### 3.4 动态提示生成

动态提示根据以下因素生成：

1. **当前对话状态**：咨询阶段、来访者情绪状态
2. **识别的认知扭曲**：需要解决的扭曲类型和严重程度
3. **记忆内容**：相关的历史信息
4. **CBT技术选择**：适合当前情况的CBT技术

提示模板示例：

```
你是一位CBT咨询师。来访者刚刚说："{user_statement}"。

从记忆中检索到的相关信息：
{retrieved_memory}

识别的认知扭曲：
{cognitive_distortions}

请使用{selected_cbt_technique}技术，以{resolution_stage}阶段的方式回应。
```

## 4. 实验设置

### 4.1 数据集

我们使用Character.ai服务创建了评估数据集：

- 选择了8个知名人物的模拟角色
- 每个角色进行5轮咨询对话
- 总共40个对话样本

### 4.2 评估指标

使用GPT-3.5作为评估器，基于以下五个标准：

1. **相关性**：响应是否与来访者陈述相关
2. **专业性**：是否使用适当的CBT技术
3. **同理心**：是否表现出理解和关怀
4. **有效性**：是否有助于解决认知扭曲
5. **自然性**：对话是否流畅自然

### 4.3 基线模型

我们比较了以下基线模型：

1. **标准GPT-3.5**：没有特殊提示的普通对话模型
2. **CBT提示的GPT-3.5**：使用静态CBT提示
3. **其他心理咨询模型**：现有的心理咨询对话模型

---

## 5. 结果

### 5.1 主要发现

CoCoA在五个评估指标上均显著优于基线模型：

- 相关性：+15%提升
- 专业性：+22%提升
- 同理心：+12%提升
- 有效性：+18%提升
- 自然性：+8%提升

### 5.2 记忆系统效果

记忆系统的使用显著提高了响应的连贯性和个性化程度：

- 基本记忆使个性化程度提高25%
- 认知扭曲记忆使问题解决效率提高30%

### 5.3 动态提示效果

动态提示相比静态提示：

- 提高了CBT技术应用的适当性
- 使响应更加情境化
- 减少了重复和模板化的响应

## 6. 讨论

### 6.1 技术贡献

CoCoA的主要技术贡献包括：

1. **专门化的记忆系统**：针对心理咨询优化的记忆结构
2. **认知扭曲的自动识别**：使用LLM识别心理治疗中的关键概念
3. **动态提示生成**：根据对话状态自适应调整的提示策略

### 6.2 实际意义

CoCoA的实际应用价值：

1. **可扩展的心理健康服务**：可以服务更多需要心理咨询的人群
2. **一致性保障**：避免人类咨询师的个体差异
3. **持续学习**：可以通过更多对话数据不断改进

### 6.3 局限性

当前系统的局限性：

1. **依赖预训练LLM**：受限于基础模型的能力
2. **评估的主观性**：心理咨询效果难以客观量化
3. **伦理考虑**：需要谨慎处理敏感心理健康问题

## 7. 结论与未来工作

### 7.1 结论

我们提出了CoCoA，一个基于CBT的对话咨询智能体，它使用专门化的记忆系统和动态提示来提供有效的心理咨询。实验结果表明，CoCoA在多个评估指标上显著优于基线模型，证明了所提出方法的有效性。

### 7.2 未来工作

未来的研究方向包括：

1. **多模态交互**：整合语音、面部表情等非语言信息
2. **个性化适应**：根据用户特点调整咨询风格
3. **长期效果评估**：跟踪咨询的长期影响
4. **伦理框架开发**：建立AI心理咨询的伦理指南

## 8. 对HeartRule的启示

### 8.1 可借鉴的技术

CoCoA的研究为HeartRule提供了以下技术启示：

#### 记忆系统设计

- **分层记忆结构**：CoCoA的基本记忆和认知扭曲记忆分离的设计值得借鉴
- **向量检索优化**：高效的记忆检索机制可以提升系统性能
- **记忆更新策略**：动态更新记忆内容保持信息新鲜度

#### 认知扭曲处理

- **自动识别机制**：LLM驱动的认知扭曲识别可以集成到HeartRule中
- **优先级排序**：根据严重程度和影响范围确定解决顺序
- **解决状态跟踪**：跟踪每个认知扭曲的解决进度

#### 动态提示策略

- **情境感知提示**：根据对话状态动态调整提示内容
- **技术选择逻辑**：基于问题类型选择最合适的咨询技术
- **阶段适应性**：根据咨询阶段调整对话策略

### 8.2 实现建议

基于CoCoA的研究，对HeartRule的实现建议：

1. **记忆系统增强**

   ```yaml
   memory_system:
     basic_memory:
       storage: vector_database
       content_types: [personal_info, key_insights, preferences]
     cognitive_memory:
       storage: specialized_store
       content_types: [distortions, patterns, triggers]
     retrieval_strategy: hybrid_semantic_keyword
   ```

2. **认知扭曲处理流程**

   ```
   用户陈述 → 提取关键短语 → 匹配扭曲模式 → 评估严重程度 → 确定优先级 → 存储到记忆 → 规划解决策略
   ```

3. **动态提示模板**

   ```python
   def generate_dynamic_prompt(context):
       memory = retrieve_relevant_memory(context)
       distortions = identify_cognitive_distortions(context.user_statement)
       technique = select_cbt_technique(distortions, context.stage)
       stage = determine_resolution_stage(context.progress)

       return f"""
       作为CBT咨询师，针对用户陈述："{context.user_statement}"

       相关记忆：{memory}
       识别扭曲：{distortions}
       使用技术：{technique}
       解决阶段：{stage}

       请生成适当的咨询回应。
       """
   ```

### 8.3 差异化设计

HeartRule与CoCoA的差异化设计考虑：

1. **更广泛的咨询方法**：不仅限于CBT，整合多种咨询流派
2. **脚本驱动架构**：使用YAML脚本定义咨询流程，提供更强的可控性
3. **人机协同模式**：支持咨询师与AI的多种协作模式
4. **渐进规则化**：从具体案例逐步抽象规则，避免过度工程化

### 8.4 研究价值

CoCoA研究对HeartRule的研究价值：

1. **验证了AI心理咨询的可行性**：证明了LLM在专业心理咨询中的应用潜力
2. **提供了技术基准**：建立了评估AI心理咨询系统的参考标准
3. **揭示了关键挑战**：识别了记忆管理、提示工程等关键技术挑战
4. **指明了改进方向**：为后续技术优化提供了明确方向
