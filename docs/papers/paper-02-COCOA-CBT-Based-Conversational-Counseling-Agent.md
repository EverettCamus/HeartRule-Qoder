## 摘要

对提供心理保健的对话智能体的需求持续增加。在本工作中，我们开发了一个称为CoCoA的心理咨询智能体，它应用认知行为疗法（CBT）技术来识别和解决来访者陈述中固有的认知扭曲。具体而言，我们构建了一个记忆系统，以有效管理咨询所需的信息，同时从来访者的陈述中提取高水平的洞察。此外，为了确保咨询智能体生成适当的响应，我们引入了动态提示，以灵活地应用CBT技术并促进信息的适当检索。我们让CoCoA与Character.ai的角色进行对话，创建了评估数据集。然后，我们请GPT评估构建的咨询数据集，我们的模型表现出与其他模型统计学上的显著差异。

## 1. 引言

在当今世界，越来越多的人表达对心理健康的担忧。因此，对解决这些问题的心理辅导需求更高。然而，有些人由于成本和社会耻辱等因素无法获得适当的心理健康护理。在这种情况下，用于心理咨询的对话智能体可以为那些对传统心理健康服务犹豫不的人来说是一个有价值的替代方案。此外，它可以通过增强可及性和便利性来有效满足日益增长的心理健康需求。

最近的研究开发了利用对话智能体改善精神状态的方法，如移情对话生成和情绪支持对话。然而，上述研究专注于提供情绪支持，而不是推进解决根本心理问题。为了解决这个问题，我们提出了一个基于LLM的咨询师智能体，它利用CBT技术提供心理咨询。CBT技术旨在识别和转换认知扭曲，即负面情绪的根本原因。

为了利用CBT技术提供类似人类的咨询，我们提出了几种方法。咨询师需要从与来访者的对话中提取相关信息以生成响应。为了有效地从来访者的陈述中提取、存储和检索高级信息，我们将记忆结构化为两个组件：用于存储通过咨询学习的个人信息的基本记忆，以及用于存储从陈述中识别的认知扭曲的认知扭曲记忆。此外，咨询师在响应之前会根据来访者的陈述规划要解决的认知扭曲并选择CBT技术及其相应的解决阶段。我们动态组装提示，使我们的咨询智能体经过此过程后生成响应。因此，我们引导智能体在生成咨询话语之前经过中间步骤以进行思维链。

为了评估我们智能体的表现，我们在CoCoA与模拟知名人物的生成智能体之间进行咨询会话。我们通过Character.ai服务选择了8位知名人物的模拟角色，如梵高或杰伊·盖茨比。然后，我们指示GPT-3.5根据五个标准评估构建的数据集。

## 2. LLM作为咨询师：CoCoA

### 2.1 概述

在本节中，我们呈现了称为CoCoA的咨询师智能体。如图1所示，我们的框架由两个组件组成：1) 记忆管理，和2) 提示构成。在构建提示时，从记忆中检索所需信息并相应使用。构建的提示作为输入提供给LLM，记为f_LLM。

**记忆管理**。智能体将咨询会话期间收集的信息存储在记忆中。每次收到来访者陈述时，系统从中提取洞察并存储在基本记忆中。如果在对话过程中识别到任何认知扭曲，它们会被保存在认知扭曲记忆中。

**提示构成**。我们将提示结构化为静态提示S和动态提示D。静态提示随着对话进行保持不变，提供一致的参考。另一方面，动态提示为每个回合定制，从外部来源检索知识，以生成适合当前情况的上下文相关提示。采用这种方法是为了灵活应用认知行为疗法（CBT）技术，并促进技术应用过程中所需信息的适当检索。

### 2.2 记忆管理

在咨询会话期间，LLM发现隐藏在来访者陈述中的信息，并将其存储为两种类型的记忆：基本记忆和认知扭曲记忆（CD记忆）。智能体将认知扭曲记忆与基本记忆分开管理和存储，以识别跨多个情节的核心信念并整合它们以解决认知扭曲。

基本记忆存储从陈述中提取的高级洞察，而CD记忆存储与认知扭曲相关的信息。当给出陈述时，LLM分析来访者所做的每个陈述以识别任何认知扭曲。如果检测到认知扭曲，（1）认知扭曲类型，（2）陈述，以及（3）Likert量表1-5分，表示扭曲的严重程度，会被追加到CD记忆中。

在构建CD记忆之后，智能体决定在现有认知扭曲中优先处理哪种类型的认知扭曲。为了决定要治疗的认知扭曲类型，它考虑三个主要组成部分：近期性、频率和严重性。近期性为最近提到的认知错误分配更高的分数，使用关于回合数的指数衰减函数。频率计算每种认知扭曲的发生次数。严重性由CD记忆中存储的每个认知扭曲的1-5分Likert量表分数决定。最终分数是通过将近期性、频率和严重性分数归一化到0-1之间的值，然后使用加权系数组合它们来计算的。

### 2.3 提示构成

#### 2.3.1 静态提示

静态提示由任务描述Task、ESC技术的解释ESC和最近的陈述U组成。因此，静态提示可以表示为：

```
Static = Task + ESC + U
```

#### 2.3.2 动态提示

动态提示由三个主要过程组成：（1）确定最佳CBT技术，（2）指定要应用的CBT阶段，（3）基于这些决定生成响应。

**CBT技术确定**
系统从基本记忆和CD记忆中检索与cd*相关的记忆。在这个过程中，我们使用Contriever，通过嵌入记忆并计算它们之间的相似性来检索记忆。要应用的CBT技术t*是从给定的CBT技术列表T中根据检索到的记忆Mr选择的。

**CBT阶段确定**
在后续步骤中，智能体的任务是从所选CBT技术t\*的各种阶段中选择一个具体的阶段S。首先，检索所选CBT技术中要采取的阶段。随后，通过参考Log（指示对话中应用的CBT技术阶段的当前进度）来确定要应用的CBT技术阶段。同时，生成当前阶段适当陈述的示例。

**响应生成**
动态提示由选定的CBT技术、CBT的适用阶段和适当的话语示例构成。然后将动态提示与静态提示集成以形成LLM的输入，从而生成响应。

## 3. 实验

### 3.1 响应生成评估

为了评估CoCoA作为咨询师的表现如何，有必要让CoCoA与对话者进行对话，并将其对话数据集与其他LLM的数据集进行比较。为此，我们促进了咨询智能体和扮演来访者的对话者智能体之间的对话，以生成咨询对话数据集。随后，我们请GPT-3.5-turbo使用0-6的量表根据五个标准对构建的数据集进行评分。

### 3.2 评估数据集构建

我们选择使用Character.ai聊天机器人服务来扮演来访者的角色。Character.ai是一种对话智能体服务，允许您与关于名人创建的模拟角色进行对话。我们从Character.ai中抽取了8个模拟角色，并与它们进行对话。我们将我们的模型与三个基线进行比较：GPT-3.5-turbo、LLAMA-2-7B-chat和Vicuna-7B。

### 3.3 评估标准

评估标准由五个元素组成，源自Blackburn等人（2001）和Shao等人（2023）的研究。其中三个与CBT技术应用的有效性、适当性和准确性相关，一个与情绪支持对话的执行相关，另一个与模型的稳定性相关。每个标准使用0-6的量表进行评分。

- **CBT有效性**：模型根据有效推理选择最佳技术的能力，考虑目标功能失调思维和来访者的核心问题。
- **CBT适当性**：模型应保持合作态度，专注于让来访者探索他们的问题，而不是在使用CBT技术时进行辩论或说服。
- **CBT准确性**：模型应准确且熟练地使用所选的CBT技术。
- **ES适当性**：模型应进行保持在先前对话或一般常识上下文中的对话，提供适当的同理心和安慰。
- **稳定性**：模型即使面对增量输入的变化，也应在相对较长的对话中保持稳定和一致。

### 3.4 评估结果

图2展示了根据五个标准评估每个模型输出并计算平均值的结果。与Vicuna-7B相比，我们的模型在所有标准上都表现更好。与其他模型相比，我们的模型在CBT有效性和准确性方面总体表现较高。然而，在ES适当性或稳定性方面没有显著差异。根据T检验结果，CoCoA在0.05水平上除稳定性外的其他标准都表现出统计学上的显著差异。

在实验过程中，其他模型获得了整个对话历史，而CoCoA只获得了直接的前一句陈述并从其记忆中检索必要的信息。尽管存在这种差异，我们模型强劲的表现可归因于专门为咨询构建的记忆系统。实验结果表明，我们的模型有效地构建了记忆并实施了适当的CBT技术。

## 4. 相关工作

将对话智能体引入心理健康领域的工作一直在进行。自然语言处理（NLP）技术导致了面向任务的医疗保健对话系统的发展。虽然该领域计算机科学的研究集中在技术方面，但其他学科探索了心理健康对话智能体在精神病学中的可用性和用户接受度。

传统上，以前的研究使用了基于规则或基于检索的模型。然而，最近的研究利用大型语言模型来改善心理健康对话智能体中的对话生成任务。

---

## 5. 结论

在本研究中，我们通过提出构建认知扭曲记忆的方法和动态构建提示以基于CBT技术生成响应，探索了开发咨询智能体的潜力。我们进行了评估模型响应的实验，展示了在CBT准确性和有效性方面的出色表现。由于缺乏来自人类来访者的咨询数据集，我们选择使用Character.ai。为了解决Character.ai用于咨询的局限性，未来的研究应该开发具有多样化和认知扭曲的角色来模拟人类来访者。

## 附录A：CoCoA算法

### 算法1：记忆管理

**输入**：pn，来访者的第n个陈述；f_LLM，LLM模型

**输出**：MB，基本记忆；MCD，CD记忆

1. MB ← 从pn中提取洞察
2. 如果pn中有cd：
   MCD ← 添加(cd, pn, 严重程度分数)
3. 返回MB, MCD

### 算法2：提示构成

1. cn+1 ← 咨询师的下一个响应
2. 如果MCD为空：
   提示 ← 静态
3. 否则：
   cd* ← 从CD中选择得分最高的cd
   Mr ← 检索与cd*和U相关的记忆
   t* ← 选择CBT技术
   stage, example ← 确定阶段
   提示 ← 静态 + 动态(t*, stage, example)
4. cn+1 ← f_LLM(提示, U)
5. 返回cn+1

## 附录B：认知行为疗法（CBT）技术

### B.1 概述

CBT是一种心理治疗方法，探索精神障碍中的认知因素并提出有效的干预方法。个体通过源于核心信念的自动思维来看待世界。负面核心信念通常导致夸大或扭曲现实，导致各种认知扭曲。在CBT技术中，采用特定的治疗策略来修改和重建认知扭曲。

### B.2 认知扭曲的类型

- **全或无思维**：将经历分为黑白对错两类
- **过度概括**：根据有限部分对整体做出快速判断
- **心理过滤**：过滤掉积极元素，过度关注情境的消极方面
- **个人化**：即使没有根据也将外部事件归因于自己
- **贴标签**：使用过度消极的语言描述自己或他人
- **读心术**：基于自己的感知或解释假设他人的想法、感受或意图

### B.3 CBT策略

我们将CBT技术类型分为20种并进行研究。以下是每种技术的简要描述。

#### B.3.1 认知重构

- **引导发现**：涉及治疗师通过提问、探索和反思引导来访者探索和理解他们的思维、情感和行为模式
- **效能评估**：帮助个体评估他们的思维或信念的有用性，分析它们在现实生活中的实用性或损害程度
- **饼图技术**：用于过度自责或责任的个体，可视化表示特定事件或结果中各种因素的贡献
- **替代视角**：询问来访者在类似情况下其他人可能如何思考，鼓励考虑不同的解释
- **去灾难化**：旨在通过评估恐惧结果的实际可能性并准备应对策略来减少想象最坏情况的倾向
- **量表问题**：要求来访者在0-10的量表上评估他们的情感或问题，有助于自我意识和视角
- **苏格拉底式提问**：深入探索来访者的思维和信念，鼓励批判性检查和考虑替代观点
- **利弊分析**：分析特定思维或信念的优势和劣势，促进更平衡的评估
- **思维实验**：鼓励来访者想象如果发生不同结果他们的思维可能会如何变化，促进思维的灵活性
- **基于证据的提问**：引导来访者找到支持或反驳他们思维的证据，促进更基于证据的思维方法
- **现实测试**：探索来访者的思维与现实的一致性，帮助他们区分思维和实际体验
- **连续体技术**：将来访者的经历置于两种极端情况之间，鼓励对情境进行更细致的评估
- **改变规则到愿望**：用现实的希望或愿望取代严格的规则或任意态度
- **行为实验**：在特定情境中尝试新行为以挑战和修改负面信念

#### B.3.2 行为激活

- **活动安排**：通过日程管理和计划积极活动来组织活动
- **问题解决技能训练**：学习解决问题情况的系统方法

#### B.3.3 自我肯定训练

- **角色扮演和模拟**：在咨询会话中模拟各种情况来练习自我肯定行为
- **自信对话技能练习**：练习自信对话技能，包括使用"我"的信息、清晰直接的语言和非语言交流

#### B.3.4 暴露

- **系统性暴露**：逐步暴露导致恐惧或焦虑的情况，让个体在学习如何管理的同时体验焦虑
- **安全行为消除**：旨在减少或消除用于应对焦虑的行为
